# AlgoDS

Open any folder in any IDE(Preferably VSCode) and type the below commands in the terminal 

Steps:
1. Clone the repo `git clone https://github.com/tejasayya/sorting.git`
2. `javac sorting/*.java`
3. `java sorting.main`
